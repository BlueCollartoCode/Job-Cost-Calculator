# Job Cost Calculator

A Python terminal app that generates professional job quotes 
for metal fabrication shops.

## Features
- Design, laser cutting, and welding cost tracking
- Hourly and flat rate pricing options
- Hours or minutes time input
- Rush job upcharge
- Repeat customer discount
- Tax exemption handling
- Variable profit margin
- Formatted professional quote output

## How to run
python Job_Cost_Calculator.py
